# LOTTO-SCRAPER-PH-NODEJS

## Setup
```shell
# Clone the repo
git clone https://github.com/mxaddict/lotto-scraper-ph-nodejs.git

# Move to the working dir
cd lotto-scraper-ph-nodejs

# Install packages
npm install
```

## Running the scraper
```shell
# Move the the working dir
cd lotto-scraper-ph-nodejs

# RUN WITH IT!
./run
```
